Section 05

Block Counting

Look at a numbered block in a 3D stack. Count how many other blocks it directly touches (shares a face with). This tests 3D spatial reasoning — a key skill for aviators.

Questions
Time per question
RULES
A block "touches" another only if they share a full face — not edges or corners. Count all 6 possible directions: front, back, left, right, top, bottom. Touching the ground does NOT count as a block.
